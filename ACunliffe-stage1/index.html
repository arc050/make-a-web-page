The Basics of The Web and HTML

The web is a collection of HTML documents. HTML stands for Hypertext Markup Language. HTML is the main type of document on the web. The major pieces to the web start with us on our browser. The browser makes HTTP requests to the internet that goes to the server and the server sends back the web page you wish to browse through the internet.

HTML is made up of:

    text content: what you see
    markup: what it looks like
    references to other documents: i.e..images & videos
    links to other pages: websites, compose email

HTML Tags:

    b = bold
    em = emphasis/italics

HTML Attributes:

    href: the value of an anchor tag that links the media to the html document.
    src: the value of an image tag that links the image to the html document.
    alt: the value of the alternate text displayed in the image tag if the image won't load.
    Image tags are called void tags since they don't have any content which is why they don;t require a closing tag.

Whitespace:

    br = allows you to create a new line of text in html.
    p = allows you to create a paragraph of content in html.

Inline and Block Elements

    A, span, br, img, and strong are all inline elements because they don't generate a box around the content like a block elements does when you use p, div, or form.

Creating a Structured Document with HTML

    HTML is the standard markup language used to create webpages.
    CSS stands for Cascading Style Sheets. CSS is the language for how the page looks and is styled.
    DOM stands for Document Object Model. DOM is a cross-platform and language independent convention for representing and interacting with the elements in HTML. DOM gives the elements in HTML the tree like structure.

Adding Style to Structure with CSS

    CSS are rules for an HTML document to abide by that changes the format and style of all the elements in the page.
    Selectors in the CSS are how you can change the rules to a specific element in a webpage.
    A declaration is a line of code that controls the "background-color"(property) and the color "black"(value of that property).
    Adding a class selector allows you to be more specific when dealing with multiple elements that are the same and want to style them separately.
    To add a CSS style page to your html document you have to reference it in the head of your webpage.
